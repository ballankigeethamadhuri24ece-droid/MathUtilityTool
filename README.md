# MathUtilityTool

This Python project is a **Mathematical Utility Tool** using a single reusable class `MyMath`. It allows students to perform multiple mathematical operations dynamically based on user input, including:

- Sum of first n natural numbers  
- First n prime numbers  
- Fibonacci series of n numbers  
- Factorial of a number  

---

## 📂 Repository Files

- **`profile.py`** – The main Python program containing the `MyMath` class and implementation of all operations.  
- **`output3.txt`** – Sample output generated by the program.  
- **`Output (3).png`** – Screenshot of the program output.
